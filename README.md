# Ava AI Assistant 🤖

Ava is an AI-powered chatbot assistant built for Avatar — an AI & Cyber Security training platform.

## 🚀 Features

- AI-powered chatbot using Groq API
- FastAPI backend
- Interactive frontend UI
- Real-time AI responses
- Context-aware conversation
- Modern responsive design

## 🛠️ Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- FastAPI
- Python
- Groq API
- Uvicorn

## ⚡ Setup Instructions

### 1. Clone Repository

```bash
git clone https://github.com/yourusername/avatar-ai-chatbot.git
```

### 2. Install Backend Dependencies

```bash
pip install -r requirements.txt
```

### 3. Add Environment Variables

Create `.env`

```env
GROQ_API_KEY=your_api_key_here
```

### 4. Run Backend

```bash
uvicorn main:app --reload --port 8000
```

### 5. Run Frontend

```bash
py -m http.server 5500
```
